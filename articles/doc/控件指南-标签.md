# 标签（Label）

**功能说明**

单行文本显示控件。

**HTML**
```html
<label id="label2"></label>
```
**属性列表**

| 属性名 | 属性值 | 说明 |
| ------------ | ------------ | ------------ |
| text |   | Label标签上显示的内容 |
| innerhtml |   | 支持html格式 |
| bindfield |   | 控件绑定字段 |

**支持公共属性**
*参见公共属性说明*

**事件列表**

| 事件名 | 说明 |
| ------------ | ------------ |
| 无 | 无 |

**支持公共事件**
*参见公共事件说明*

# 实例

## 1、获取或修改label显示的文字

```javascript
$id("xxx").get("text");//可以获取到（单行label）
$id("xxx").set("text", "单行label")
```
