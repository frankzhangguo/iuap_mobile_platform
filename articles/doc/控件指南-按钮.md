# 按钮（Button）

**功能说明**

Button 控件为按钮，用于响应用户的点击事件。

**HTML**

```html
<input id="button0" value="按钮" onclick="onclick()" type="button"/>
```

**属性列表**

| 属性名 | 属性值 | 说明 |
| ------------ | ------------ | ------------ |
| value | string | 按钮显示文字 |
| bindfield |   | 控件绑定字段 |

**支持公共属性**
*参见公共属性说明*

**事件列表**

| 事件名 | 说明 |
| ------------ | ------------ |
| 无 | 无 |

**支持公共事件**
*参见公共事件说明*

# 实例

## 1、value与onclick事件的同时使用

```html
<input id="button0" value="按钮" onclick="onclick()" type="button"/>
```

```javascript
onclick(sender, args) {
	alert($id("button0").get("value")); //button上显示“按钮”
}
```
